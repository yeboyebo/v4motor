Adds your Google AdWords Remarketing JavaScript tracking code to your WordPress website. 

== Description ==

Adding Google AdWords Remarketing code to your website has never been easier. Simply copy and paste your AdWords Remarketing code and that's it. 

Why use Google AdWords Remarketing? 

* Target visitors who visited your website before;
* Create customized messages for specific visitors;
* ...and more.

For more information visit: [Google AdWords](http://www.google.com/adwords) or [OnlineAds.lt](http://onlineads.lt).

== Installation ==

1. Upload `google-adwords-remarketing` directory to the `/wp-content/plugins/` directory or use plugin search - Admin > Plugins > Add new > Search for 'Google AdWords Remarketing'.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Copy Remarketing code from Google AdWords and paste it in the settings page (Admin > Settings > Google AdWords Remarketing).
4. Click 'Save' and it's done!
